# Local School Setup

Use this when a school wants the system to run inside the school without internet.

1. Install Python on the main school computer.
2. Open the project folder.
3. Double-click `start-local.bat`.
4. Open `http://127.0.0.1:5000` on the main computer.
5. Other computers on the same Wi-Fi or LAN can open the system using the main computer IP address, for example `http://192.168.1.20:5000`.

The main computer must stay on while teachers and students are using the system.

Back up these regularly:

- `smart_schools_sms.db`
- `uploads/`
