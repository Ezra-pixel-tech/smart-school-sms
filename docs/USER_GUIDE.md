# Smart Schools SMS User Guide

## System Admin

- Login with the system admin account.
- View all registered schools.
- Confirm schools have completed their profile setup.

## School Admin

- Register the school and login.
- Complete the school profile with name, motto, crest, address, phone, email, academic year, and term.
- Create classes, subjects, teachers, and students.
- Give each student their unique username and password.
- Record attendance and fee payments.
- Publish school notices.
- Build the school timetable.
- Add library resources.

## Teacher

- Login with the username and password created by the school admin.
- View students, classes, subjects, timetable, notices, and library resources.
- Enter student scores for each subject.

## Student

- Login with the unique username and password created by the school.
- View notices, timetable, library resources, and personal results.
- Print the report sheet or save it as a PDF for parents.
